About

From Software Engineer to Cloud Engineer

A summary of where I've been, and where I'm headed.

I'm a Software Engineer with 2.5 years of hands-on experience designing and managing CI/CD pipelines on Azure DevOps, automating deployments, and running containerized workloads on Kubernetes — primarily within the BFSI / Insurance domain at Persistent Systems.

My day-to-day work sits at the intersection of software engineering and cloud infrastructure: building multi-stage release pipelines, containerizing applications with Docker, orchestrating them on AWS EKS with Helm, and tightening the feedback loop between development and production.

What started as a software engineering role has grown into a genuine pull toward cloud computing. I'm continuously upskilling in Microsoft Azure and DevOps practices, with the long-term goal of becoming a dedicated Cloud Engineer — someone who designs the infrastructure other teams build on.
